Данная программа принимает на вход список, элементы которого могут быть любыми. В данном массиве она находит самую длинную строку, если таких будет несколько, то выбирает первую. В найденной строке находит самую длинную подстроку без повторений. Таких строк может быть несколько, но возвращает функция первую подстроку. Результат функции - два объекта: индекс самой длинной строки списка и самая длинная подстрока без повторений. Если в исходном списке нет строк, то программа возвращает индекс -1 и пустую строку.

для запуска программы нужно вызвать функцию main следующим образом:

list = ["Ваш список"]
